# CLI API

*Warning: documentation is work in progress*

This command-line interface (CLI) provides tools to control and configure
SmartVNS devices over a wired connection (USB/cable). Use the CLI to
set/get system configurations, set the internal datetime, and pair devices.

The CLI is installed in the environment under the name `smartvns-cli`
